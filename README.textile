h1. Financoid

h2. Info

Financoid helps the user keep track of her everyday expenses by recording monetary transactions, while built-in statistical and data visualiztion tools provide an overview of monthly 
expenditures and expense patterns through time (eg. which month or product category seems to be the most costly).

h2. Features

* Financial account overview
* Recent purchases screen
* Management and profiling of everyday expenses
* Monthly and by-product-category statistics
* Export of data to CSV/XLS/SQL format
