h1. Financoid

h2. Info

This application provides the user with tools for keeping track of her everyday expenses,
while at the same time provides statistical and data visualiztion tools for monthly
expenditures and expense patterns through time (eg. which month or product category
seems to be most expense-heavy).

h2. Features

* Financial account overview
* Recent purchases screen
* Management and profiling of everyday expenses
* Monthly and by-product-category statistics
* Export of data to CSV/XLS/SQL format
